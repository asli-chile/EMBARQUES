# ASLI Web2 (sitio público Next.js)

Este directorio es una **aplicación aparte** del sistema Embarques (Astro en la raíz del repo).

En **Vercel** debe existir un **proyecto propio** con **Root Directory = `web2`**. Un solo proyecto enlazado solo a la raíz del repo **no** despliega esta app.

Instrucciones detalladas: [docs/DESPLEGUE-VERCEL.md](../docs/DESPLEGUE-VERCEL.md).

Variables de entorno: copiar de `.env.example` (definir en Vercel → Settings → Environment Variables).
